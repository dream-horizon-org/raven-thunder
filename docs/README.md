# Thunder Documentation

This folder contains the Docusaurus documentation site for Thunder.

## Quick start

1) Install dependencies
   - Requires Node.js 18+ and npm or yarn
   - Run:
     - npm install
     - npm run start

2) Develop locally
   - Starts a local dev server at http://localhost:3000/

3) Build static site
   - npm run build
   - Output in build/ (can be hosted on any static host or GitHub Pages)

## Structure
- docs/: Markdown/MDX content
- src/pages/: Landing page and custom pages
- static/: Static assets (images, etc.)
- sidebars.js: Sidebar structure
- docusaurus.config.js: Site config
