Thunder is a Java 17 + Vert.x platform for managing CTAs (Call‑to‑Actions), Nudges, and Behaviour Tags.

- Multi‑module: `thunder-core`, `thunder-api`, `thunder-admin`
- Aerospike integration, health checks, Docker Compose dev environment
- CI with unit/integration tests; release as Docker images and fat JARs
